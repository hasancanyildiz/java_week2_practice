# Library Console App

A small Java project I made to practice Stream API and lambda expressions.

## What it does

The app works with a list of 18 books. Each book has a title, author, year, genre, and availability status. It used Java Streams to filter, sort, group, and get statistics from this list.

## What I practiced

- Filter books by author
- Sort by year and genre
- Get titles of available books
- Group books by genre
- Count books per genre
- Calculate average publish year
- Print a genre report with nested streams

## How to run

Open in IntelliJ, run `Main.java`, and check the console output.
